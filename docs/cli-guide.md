# Command Line Interface Guide

The PASA Research Fetcher provides a powerful command-line interface for searching and downloading research papers.

## Basic Usage

```bash
pasa-fetcher search "your research query"
```

## Commands

### `search` - Search for Papers

Search for research papers using natural language queries.

```bash
pasa-fetcher search "machine learning transformers" [OPTIONS]
```

### `search-complete` - Complete Search with Guarantee

Search for research papers with completion guarantee and automatic relevance sorting. This command ensures the search runs until fully complete, returning all available papers sorted by relevance score.

```bash
pasa-fetcher search-complete "machine learning transformers" [OPTIONS]
```

**Standard Search Options:**

- `--max, -m INTEGER`: Maximum number of results to return (default: 50)
- `--output, -o PATH`: Output directory for downloads
- `--format, -f [json|table]`: Output format (default: json)
- `--pdfs/--no-pdfs`: Download PDF files (default: no-pdfs)
- `--tex/--no-tex`: Download TeX source files (default: no-tex)
- `--timeout INTEGER`: Request timeout in milliseconds (default: 30000)

**Complete Search Options:**

All standard options plus:

- `--sort/--no-sort`: Sort by relevance score (default: sort)
- `--timeout INTEGER`: Extended timeout (default: 60000ms for complete search)

**Examples:**

```bash
# Basic search with JSON output
pasa-fetcher search "attention mechanism neural networks"

# Search with table output (human-readable)
pasa-fetcher search "computer vision" --format table --max 10

# Search and download PDFs
pasa-fetcher search "deep learning" --pdfs --output ./papers --max 5

# Search with custom timeout
pasa-fetcher search "quantum computing" --timeout 60000 --max 20

# Complete search with guaranteed results and relevance sorting
pasa-fetcher search-complete "machine learning transformers" --format table --max 15

# Complete search with downloads and custom sorting
pasa-fetcher search-complete "neural networks attention" --pdfs --output ./complete-papers --sort

# Complete search without sorting (faster)
pasa-fetcher search-complete "computer vision" --no-sort --max 20
```

### `config` - Show Configuration

Display current configuration settings.

```bash
pasa-fetcher config
```

### `version` - Show Version

Display the package version.

```bash
pasa-fetcher version
```

### JSON Format (Default)

Returns detailed paper information in JSON format:

```json
[
  {
    "metadata": {
      "arxiv_id": "2301.00001",
      "title": "Paper Title",
      "authors": [{"name": "Author Name", "affiliation": "Institution"}],
      "abstract": "Paper abstract...",
      "published_date": "2023-01-01T00:00:00",
      "categories": ["cs.CV", "cs.AI"],
      "primary_category": "cs.CV"
    },
    "pdf_url": "https://arxiv.org/pdf/2301.00001.pdf",
    "arxiv_url": "https://arxiv.org/abs/2301.00001",
    "relevance_score": 0.95,
    "summary": "PASA-generated summary..."
  }
]
```

## Example Queries

Try these example queries to test different research areas:

```bash
# Computer Vision
pasa-fetcher search "object detection YOLO" --format table --max 10

# Natural Language Processing  
pasa-fetcher search "large language models GPT transformer" --pdfs --output ./nlp-papers

# Machine Learning Theory
pasa-fetcher search "optimization algorithms gradient descent" --format table

# Specific Techniques
pasa-fetcher search "attention mechanism neural networks" --max 15

# Interdisciplinary
pasa-fetcher search "machine learning healthcare medical diagnosis" --pdfs --max 5
```

## Advanced Usage

### Combining with Shell Commands

```bash
# Count papers by author
pasa-fetcher search "deep learning" --max 100 | jq '.[].metadata.authors[].name' | sort | uniq -c | sort -nr

# Extract just titles
pasa-fetcher search "computer vision" | jq -r '.[].metadata.title'

# Filter by relevance score
pasa-fetcher search "neural networks" | jq '.[] | select(.relevance_score > 0.8)'
```

### Scripting

Use in bash scripts for automated paper collection:

```bash
#!/bin/bash
topics=("machine learning" "computer vision" "natural language processing")

for topic in "${topics[@]}"; do
    echo "Fetching papers for: $topic"
    pasa-fetcher search "$topic" --pdfs --output "./papers/$topic" --max 20
done
```
